# gel extension list-available

List available extensions.

```cli-synopsis
gel extension list-available [<options>]
```

## Options

See Connection options for global options.

## Examples

```bash
$ gel extension list-available
┌─────────┬───────────────┐
│ Name    │ Version       │
│ postgis │ 3.4.3+6b82d77 │
└─────────┴───────────────┘
```
